## Installation
```bash
┌─[root@parrot]─[/tmp]
└──╼ # git clone https://github.com/FonderElite/nvim-config
----------------------------------------------------------
*Replace the ~/.config/nvim/init.vim with customized init.vim
┌─[root@parrot]─[/tmp]
└──╼ #cat init.vim | tee $HOME/.config/nvim/init.vim
```
